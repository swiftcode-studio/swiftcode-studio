# Swiftcode Studio - Logo Collection

## Logo Designs Created

Tôi đã tạo 4 phiên bản logo khác nhau cho Swiftcode Studio:

### 1. **logo.svg** - Logo chính với tagline
- **Kích thước**: 200x60px
- **Thiết kế**: Icon gradient + tên công ty + tagline
- **Sử dụng**: Header chính, business cards, documents

### 2. **logo-icon.svg** - Icon đơn giản (Đang sử dụng)
- **Kích thước**: 60x60px  
- **Thiết kế**: Circle gradient với chữ "S"
- **Sử dụng**: Favicon, header navigation, social media

### 3. **logo-modern.svg** - Logo Instagram-inspired
- **Kích thước**: 200x60px
- **Thiết kế**: Icon giống Instagram + tên công ty
- **Sử dụng**: Marketing materials, Instagram profile

### 4. **logo-tech.svg** - Logo automation/tech
- **Kích thước**: 200x60px
- **Thiết kế**: Flow/automation icon + tech stack
- **Sử dụng**: Technical documents, developer-focused content

## Color Scheme
- **Primary Blue**: #2563eb (Blue-600)
- **Primary Purple**: #9333ea (Purple-600)
- **Gradient**: Blue to Purple linear gradient

## Brand Identity
- **Company**: Swiftcode Studio
- **Tagline**: "Code fast. Ship smart. Grow together."
- **Specialization**: Instagram Automation & AI Solutions
- **Tech Stack**: React • Rails • AI • Automation

## Usage Guidelines
- Logo có thể scale mà không mất chất lượng (SVG)
- Sử dụng trên nền trắng hoặc nền sáng
- Đảm bảo có đủ không gian xung quanh logo
- Không thay đổi màu sắc hoặc tỷ lệ

## Files Location
Tất cả logo được lưu trong thư mục `/public/` để có thể truy cập trực tiếp từ website.
